#  Auto Emailer

A simple Python project that sends emails automatically using SMTP.  
Perfect for beginners learning how to send emails securely in Python.

---

##  Features
- Sends emails using `smtplib` and `email.message`
- Uses `.env` file to hide sensitive credentials
- Includes `.gitignore` to prevent secrets from being pushed
- Ready for GitHub deployment
